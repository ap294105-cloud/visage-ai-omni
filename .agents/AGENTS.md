3. **Oculomotor Tracking (Cognitive State)**: Reads gaze fixation, blink rate (EAR), and saccadic eye movements. Proves cognitive load.
4. **Algorithmic Colorimetry (Vascular State)**: Reads thermal proxies via exact LAB color space shifts (flush/blanching). Proves involuntary "fight or flight" autonomic responses.
5. **Pupillometry Approximation (Autonomic State)**: Reads micro-fluctuations in pupil diameter. Proves Central Nervous System arousal.

**The Fusion Mechanism: Bayesian Inference**
These 5 scores MUST NEVER be simply averaged. They must be fused using Bayes' Theorem to calculate the absolute certainty of a state given the evidence:
`P(State|Evidence) = [P(Evidence|State) * P(State)] / P(Evidence)`

This mathematical approach guarantees state certainty (0-100%).
